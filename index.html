STAC Liability and Claims Extension

Quality metadata and provenance for STAC Items and Collections

About: This extension adds liability tracking, quality metadata (ISO 19157), and provenance information (W3C PROV) to STAC Items and Collections. It supports CEOS-ARD certification and semantic web integration.

Version Compatibility

⚠️ v1.3.0 Breaking Changes: v1.3.0 introduces breaking changes for full ISO compliance. MD_Scope now requires structured objects, and QuantitativeResult values are arrays. See migration guide.

✓ v1.2.0 Full Backward Compatibility: v1.2.0 is 100% backward compatible with v1.1.0. All v1.1.0 STAC Items will validate successfully against v1.2.0 schema.

Quick Start

Add to your STAC Item:

{
  "stac_extensions": [
    "https://luciocola.github.io/stac-extension-liability-claims/v1.3.0/schema.json"
  ],
  "properties": {
    "liability:responsible_party": "European Space Agency (ESA)",
    "liability:origin": "Copernicus Sentinel-2",
    "liability:quality": { 
      "scope": { "level": "dataset" },
      "report": [...]
    },
    "liability:prov": { ... }
  }
}
