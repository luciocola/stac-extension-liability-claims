=== Example 2: RDF/Turtle Implementation

This example shows the same information in RDF/Turtle format:

[source,turtle]
----
@prefix liability: <https://stac-extensions.github.io/liability-claims/v1.1.0/schema.json#> .
@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dcterms: <http://purl.org/dc/terms/> .
@prefix prov: <http://www.w3.org/ns/prov#> .

<http://example.org/dataset/lidar-survey-2025>
    a dcat:Dataset, prov:Entity ;
    dcterms:title "LiDAR Survey 2025" ;
    dcterms:created "2025-11-15T09:00:00Z"^^xsd:dateTime ;
    
    liability:liability_framework "ISO 19157 + Professional Surveyor Standards" ;
    liability:jurisdiction "US" ;
    liability:liability_terms <https://example.org/terms/survey-services> ;
    
    prov:wasAttributedTo <http://example.org/org/xyz-surveys> ;
    prov:wasGeneratedBy <http://example.org/activity/aerial-survey-2025-11> .

<http://example.org/org/xyz-surveys>
    a prov:Organization ;
    foaf:name "XYZ Aerial Surveys Inc." ;
    liability:insurance_provider "Professional Liability Insurance Corp" ;
    liability:policy_number "PLIC-2025-789456" .
----

=== Example 3: XML Schema Implementation (ISO 19115-2)

This example shows integration with ISO 19115 metadata:

[source,xml]
----
<gmd:MD_Metadata>
  <gmd:dataQualityInfo>
    <gmd:DQ_DataQuality>
      <gmd:scope>
        <gmd:DQ_Scope>
          <gmd:level>
            <gmd:MD_ScopeCode codeList="..." codeListValue="dataset"/>
          </gmd:level>
        </gmd:DQ_Scope>
      </gmd:scope>
      
      <gmd:report>
        <gmd:DQ_AbsoluteExternalPositionalAccuracy>
          <gmd:result>
            <gmd:DQ_QuantitativeResult>
              <gmd:valueUnit>
                <gml:BaseUnit gml:id="meters">
                  <gml:identifier codeSpace="SI">meters</gml:identifier>
                </gml:BaseUnit>
              </gmd:valueUnit>
              <gmd:value>
                <gco:Record>0.15</gco:Record>
              </gmd:value>
            </gmd:DQ_QuantitativeResult>
          </gmd:result>
        </gmd:DQ_AbsoluteExternalPositionalAccuracy>
      </gmd:report>
      
      <gmd:lineage>
        <gmd:LI_Lineage>
          <gmd:statement>
            <gco:CharacterString>LiDAR collected by XYZ Aerial Surveys under professional surveyor standards</gco:CharacterString>
          </gmd:statement>
          <gmd:source>
            <!-- Extended with liability information -->
            <ext:LI_Source_Liability>
              <ext:liabilityFramework>
                <gco:CharacterString>ISO 19157 + Professional Surveyor Standards</gco:CharacterString>
              </ext:liabilityFramework>
              <ext:jurisdiction>
                <gco:CharacterString>US</gco:CharacterString>
              </ext:jurisdiction>
            </ext:LI_Source_Liability>
          </gmd:source>
        </gmd:LI_Lineage>
      </gmd:lineage>
    </gmd:DQ_DataQuality>
  </gmd:dataQualityInfo>
</gmd:MD_Metadata>
----
