== Annex C: Abstract Test Suite

=== Conformance Class: Core

==== Test /conf/core/framework

[cols="1,4",width="90%"]
|===
2+|*Test*
2+|http://www.opengis.net/spec/liability-claims/1.1/conf/core/framework
|Requirement |/req/core/framework
|Test purpose |Verify that liability metadata declares the applicable legal framework
|Test method |Inspect the liability metadata. Verify that `liability_framework` property exists and contains a non-empty string identifying the legal or regulatory framework.
|===

==== Test /conf/core/jurisdiction

[cols="1,4",width="90%"]
|===
2+|*Test*
2+|http://www.opengis.net/spec/liability-claims/1.1/conf/core/jurisdiction
|Requirement |/req/core/jurisdiction
|Test purpose |Verify that jurisdiction is properly specified
|Test method |Inspect the liability metadata. Verify that `jurisdiction` property exists and contains a valid ISO 3166-1 alpha-2 code or recognized jurisdiction identifier.
|===
